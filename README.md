# Fitspire (Expo + TypeScript)

This folder contains an Expo + TypeScript React Native starter for the Fitspire mobile app.

Quick start:

```bash
cd "fitspire-mobile"
npm install
npx expo start
```

Notes:
- This is an initial scaffold with navigation and placeholder screens.
- Install native dependencies listed in `package.json` (react-navigation, reanimated, async-storage, zustand, react-native-svg, etc.)
- After installing, run `npx expo prebuild` if you need native builds; `expo start` opens Metro for running on Android via Expo Go or emulator.
